# 🎨 カラーコード・コンバーター

色のコード（HEX, RGB, HSL）を相互に変換できる、シンプルなWebアプリです。
数値を入力すると、背景色がリアルタイムで変化し、直感的に色を確認できます。

## ✨ できること（機能）

  * **3つの形式に対応:** 以下の色コードを自動で変換します。
      * HEX (例: `#3498db`)
      * RGB (例: `52, 152, 219`)
      * HSL (例: `204, 70%, 53%`)
  * **リアルタイムプレビュー:** 入力した色が、瞬時に画面全体の背景色に反映されます。
  * **シンプルなデザイン:** 必要な機能だけをまとめた、使いやすい画面です。

## 🚀 使い方

特別なインストールは必要ありません。

1.  `color-converter` フォルダの中にある **`index.html`** をダブルクリックします。
2.  ブラウザ（ChromeやEdgeなど）が開きます。
3.  入力欄に好きな色の値を入力してください。自動で変換されます。

## 📂 フォルダの構成

```text
color-converter/
└── index.html    # アプリ本体（ここから起動します）
```

## 🛠️ 使っている技術

  * **HTML5:** 画面の骨組み
  * **CSS3:** デザインとアニメーション
  * **JavaScript:** 色の計算と変換処理
